# TaskTracker CLI

A lightweight, interactive Command Line Interface (CLI) application built in Java to efficiently manage tasks. 

This project demonstrates core Java fundamentals, including control flow, string manipulation, memory management using collections (`ArrayList`), and exception handling, all without relying on heavy external frameworks.

## 🚀 Features

Currently, the CLI operates in-memory and supports the following commands:
- **Add**: Create a new task.
- **List**: Display all current tasks with their respective IDs.
- **Update**: Modify the description of an existing task using its ID.
- **Exit**: Gracefully terminate the application.

## 🛠️ Technology Stack
- **Language:** Java
- **Build Tool:** Maven
- **Environment:** Standard console/terminal
